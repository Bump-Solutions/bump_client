# Bump Client

**Bump Client** is the official frontend application for **Bump**, built with React, Vite, and TypeScript. It provides a smooth, modern user interface for browsing products, chatting with other users, and managing your profile. Data fetching and caching are handled via TanStack Query, with HTTP requests powered by Axios.
